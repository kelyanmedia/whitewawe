# Команды для разработки и продакшена

 **УСТАНОВКА ЗАВИСИМОСТЕЙ** — Команда запуска: `npm i`

1. **Режим разработчика**
 Команда запуска: `npm run dev`

2. **Режим продакшена**
 Команда запуска: `npm run build`

3. **Режим продакшена и создание ZIP-архива с результатом**
 Команда запуска: `npm run zip`

4. **Режим продакшена без создания WEBP изображений и действий, связанных с этим форматом**
 Команда запуска: `npm run devbuild`
